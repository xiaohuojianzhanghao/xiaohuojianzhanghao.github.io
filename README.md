## 小火箭节点

下载安装好Shadowrocket之后，下一步就是配置节点了。没有节点的话，小火箭只是一款空壳应用，无法发挥实际作用。节点是Shadowrocket发挥功能的"燃料"，它决定了你的网络请求通过哪里转发。

### 节点是什么

简单来说，节点就是一个中转服务器。你的设备连接到这个服务器，然后通过它访问目标网站或应用。Shadowrocket支持多种协议，每种协议对应不同类型的节点。

### 常见协议类型

| 协议 | 特点 | 适用场景 |
|------|------|---------|
| Shadowsocks | 轻量、速度快、配置简单 | 通用场景，推荐新手选择 |
| ShadowsocksR | Shadowsocks的增强版，支持更多混淆 | 某些网络环境下更稳定 |
| VMess | 速度优秀，推荐配合WebSocket使用 | 追求速度和稳定的老用户 |
| Trojan | 伪装成HTTPS流量，隐蔽性强 | 对安全性要求较高的用户 |
| HTTP/SOCKS5 | 标准代理协议 | 特定场景的备用方案 |

### 节点的来源

节点的获取方式主要有以下几种：

**订阅链接（机场）**：这是最常见的节点获取方式。所谓的"机场"就是提供节点订阅的服务商。你购买套餐后，服务商会给你一个订阅链接，在Shadowrocket中导入这个链接，所有节点就会自动添加进来，非常方便。节点订阅到期后续费更新即可继续使用。

**自建节点**：如果你有一台海外服务器，可以自己搭建节点。这种方式需要一定的技术基础，但胜在完全掌控自己的数据。常见的搭建方案包括使用一键脚本部署Shadowsocks或V2Ray等服务端。

**免费节点**：网上有一些公开的免费节点资源。但免费节点的稳定性、速度和安全性都无法保证，不推荐用于重要场景。

### 节点怎么导入

在Shadowrocket中导入节点非常简单，这里以订阅链接为例：

1. 打开Shadowrocket应用，点击底部导航栏的"节点"标签。

4. 点击"完成"，Shadowrocket会自动拉取节点列表。
5. 在节点列表中点击你想要的节点，选中后返回首页，点击顶部的开关即可开启连接。
