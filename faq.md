## 账号使用相关

### Q: 美区 Apple ID 安全吗？
A: 完全安全。我们的账号都是全新注册，按照 Apple 的规范使用，被封禁的风险极低。我们还提供24/7的技术支持，确保您的账号安全。

### Q: 可以修改密码吗？
A: 完全可以。账号完全属于您，您有完全的控制权。建议您在首次登录后修改密码，以增强安全性。

### Q: 如何修改密码？
A: 您可以通过以下步骤修改密码：
1. 访问 Apple ID 官网（https://appleid.apple.com）
2. 使用您的 Apple ID 登录
3. 点击"安全"部分
4. 点击"更改密码"
5. 输入旧密码和新密码
6. 点击"更改密码"

### Q: 可以修改绑定的邮箱吗？
A: 可以。您可以在 Apple ID 设置中修改绑定的邮箱地址。

### Q: 可以绑定支付方式吗？
A: 可以。您可以绑定信用卡或其他支付方式来购买应用。建议您绑定自己的支付方式，以便购买应用时使用。

### Q: 账号会被封禁吗？
A: 我们的账号都是全新注册，按照 Apple 的规范使用，被封禁的风险极低。但如果您用于违法或不当用途，账号可能会被 Apple 封禁。

### Q: 账号可以用多久？
A: 美区 Apple ID 可以永久使用，没有时间限制。只要您妥善保管账号，就可以一直使用。

### Q: 一个账号可以在多少台设备上使用？
A: 一个 Apple ID 可以在最多5台 iOS 设备上使用。

### Q: 如何在 iOS 设备上登录？
A: 您可以通过以下步骤登录：
1. 打开"设置"应用
2. 点击顶部的"[您的名字]"（如果您已登录，首先点击"退出登录"）
3. 点击"没有 Apple ID 或忘记了？"
4. 选择"创建新 Apple ID"或"使用现有 Apple ID"
5. 输入您的 Apple ID 和密码
6. 按照屏幕上的指示完成登录

### Q: 如何在 App Store 中下载应用？
A: 您可以通过以下步骤下载应用：
1. 打开 App Store
2. 点击底部的"搜索"标签
3. 搜索您想要的应用（例如 Shadowrocket）
4. 点击应用图标
5. 点击"获取"按钮
6. 使用 Face ID、Touch ID 或 Apple ID 密码进行身份验证
7. 等待应用下载和安装

- 支持 Surge 配置文件

### Q: 如何续费 Shadowrocket 账号？
