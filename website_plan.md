## 5. 技术栈与部署

*   **静态网站生成器**: Jekyll
*   **部署平台**: GitHub Pages
*   **主题**: 选择一个简洁、响应式且易于定制的 Jekyll 主题。

## 6. 后续步骤

1.  安装 Jekyll 环境。
2.  选择并配置 Jekyll 主题。
3.  创建 `_config.yml` 文件。
4.  根据上述规划创建页面文件 (`.md` 或 `.html`)。
5.  编写详细内容。
6.  测试网站并部署到 GitHub Pages。
